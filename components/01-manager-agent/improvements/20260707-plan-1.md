# Plan 1 — UI & Codebase Redundancy Cleanup (2026-07-07)

- Remove legacy Dashboard UI and its associated API paths
- Wire WebSocket into workspace for real-time planner/executor feedback
- Clean up redundant action buttons and code

- Add `executionEvents` state + `pushExecutionEvent`/`clearExecutionEvents` to `sessionStore`

**Edit:**
- `frontend/src/App.tsx` — call `useWorkspaceSocket()`
- `frontend/src/stores/sessionStore.ts` — clear execution events on new message; add push/clear methods
- `frontend/src/sections/OutputSection.tsx` — add `ExecutionProgress` component showing live planner status and result details
